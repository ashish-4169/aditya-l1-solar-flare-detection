satellite.

is complete and validated.

## 5. Validated Results (Milestone 1)

**Observation date:** 2026-06-18  
**Detector:** SDD2  
**Duty cycle:** 90.9% (78 558 / 86 400 seconds valid)

| # | Class | Start (UTC) | Peak (UTC) | End (UTC) | Duration | Fluence |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| 1 | **Strong** | 11:55:18 | 11:56:49 | 11:57:29 | 132 s | 8034 cts |
| 2 | **Strong** | 22:03:20 | 22:03:27 | 22:03:38 | 19 s | 770 cts |

**Diagnostic note (why two peaks near t=28 000 s and t=54 000 s were not
detected):**  
- t≈27 916 s: 7.87σ amplitude but maximum consecutive run = 4 s (below 5 s
  threshold). Confirmed Poisson-fluctuating elevated interval, not a coherent
  flare onset.  
- t≈53 744 s: 9.78σ amplitude but run = 2 s. Single-second spike consistent
  with a cosmic ray / energetic particle hit. Correctly rejected.

## 7. Troubleshooting

| Error | Cause | Fix |
|:---|:---|:---|
| `ValueError: Big-endian buffer not supported` | FITS arrays passed directly to pandas without endianness conversion | Ensure `.astype(float)` after every `np.array(hdul[...].data[col])` call |
| `TypeError: NDFrame.fillna() got unexpected keyword argument 'method'` | Pandas 2.0 removed `method=` from fillna | Replace `fillna(method='bfill')` with `.bfill()` |
| `UnicodeEncodeError: 'charmap' codec` | Windows console (cp1252) cannot print Unicode characters | Run with `python -X utf8` or replace Unicode symbols with ASCII equivalents |
| `FileNotFoundError` for `.lc` or `.gti` | Path resolution failed | The script tries both `SDD2/file/file` and `SDD2/file.gz`; verify the `sdd2_dir` path in `main()` |
| `ModuleNotFoundError: scipy` | scipy not installed | scipy is optional; remove any import if present, pipeline does not use it |
| Empty `flare_catalog.csv` (header only) | Threshold too high or short observation | Lower `n_sigma` from 5.0 to 3.0 as a diagnostic test |

## 8. Key Design Decisions

| Decision | Rationale |
|:---|:---|
| GTI as authoritative mask (not just FITS NaNs) | NaNs in the FITS may be incomplete; GTI is the definitive quality flag from the pipeline |
| Rolling median (not mean) for background | Resistant to outliers — a 97 cts/s flare does not shift the median if it represents <50% of the window |
| MAD × 1.4826 for sigma (not std dev) | Flare peaks inflate standard deviation; MAD is bounded at the 50th percentile |
| 5-second consecutive trigger | Eliminates cosmic ray hits (1 sample) and Poisson noise bursts (2–4 samples) |
| 10-second gap bridge | Bridges counting-statistics dips within genuine flares without merging separate events |
| `spectrum = None` in FlareEvent | Forward-compatible hook for Milestone 2 spectral attachment |

## 9. Known Limitations

1. **Single day only** — pipeline is hardcoded to the 2026-06-18 observation.
   Milestone 3 should generalise to process multiple dates in a loop.

2. **No energy resolution** — the `.lc` file integrates all 340 energy channels.
   Spectral contamination from particle background vs solar X-rays is not
   separated. This is the primary goal of Milestone 2 using the `.pi` file.

3. **Classification not GOES-equivalent** — the Weak/Moderate/Strong classes
   are based on raw count rates, not calibrated flux (W m⁻²). Spectral
   response calibration is needed to convert to GOES classes.

4. **SDD1 inactive on this date** — SDD1 had zero GTI intervals. Multi-detector
   cross-validation is not possible for this observation.

5. **Background may underestimate during prolonged activity** — if the Sun
   is continuously active for >1 hour, the 7200-second window itself may
   absorb emission. Visual inspection of `flare_plot.png` is recommended.
